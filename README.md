📦 Linehaul

Ferramenta web simples para controle de pallets por unidade, com geração de relatório padronizado para operação logística.

---

🚀 Funcionalidades

- Contagem rápida por unidade (+1, +5, +10)
- Controle de unidades em doca (🚚)
- Geração de relatório formatado (pronto para WhatsApp)
- Ordenação automática por maior volume
- Observação opcional de contagem (🔥)
- Dados salvos automaticamente no navegador

---

📱 Uso

1. Abra o app no navegador
2. Atualize os valores
3. Gere o relatório
4. Copie e envie

---

📲 Instalar como app

- Abra no navegador do celular
- Toque em “Instalar app” (quando aparecer)
- Ou use “Adicionar à tela inicial”

---

🛠️ Tecnologias

- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage

---

📌 Objetivo

Agilizar a rotina operacional com um relatório claro, rápido e padronizado.
